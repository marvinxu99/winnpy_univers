Deploy to PythonAnywhere:
#https://help.pythonanywhere.com/pages/Virtualenvs
(1) mkvirtualenv myvirtualenv --python=/usr/bin/python3.8
(2) workon myvirtualenv
(3) pip install flask
(4) pip install -r requirements.txt   (generated by pip freeze > requirements.xtx)
(5) go to the Web tab and set up the Virtualenv:
     /home/winnpysoft/.virtualenvs/myvirtualenv

(6) set up settings in Winnpysoft_pythonanywhere_com_wsgi.py
